# CanadaClassi

CanadaClassi はカナダ向けのクラシファイド（掲示板）Web UI モックです。
静的HTML/CSS/JSで構成され、主要ページの画面設計・導線・UIコンポーネントを確認できます。
実装は未定であり、本リポジトリはUI/UX検討のための資料です。

## 内容
- 一覧、詳細、投稿、プロフィール、問い合わせなどの主要画面
- ブランドアセットとUIスタイル
- 学校データのサンプル（`data/`）

## 使い方
- `index.html` をブラウザで開く
- 画像・CSS・JSは同階層/`assets` を参照

## 構成
- `index.html` など: 画面テンプレート
- `style.css`: 全体スタイル
- `script.js`: 画面用の補助スクリプト
- `assets/`: 画像・ブランド素材
- `data/`: サンプルデータ

## ステータス
- 画面/導線の検討用モック（実装は未定）

# CanadaClassi (English)

CanadaClassi is a UI mock for a Canada-focused classifieds web product.
It is built with static HTML/CSS/JS to review key page layouts, user flows, and UI components.
Implementation is not decided yet; this repository is for UI/UX exploration only.

## Contents
- Core screens such as list, detail, post, profile, and inquiry
- Brand assets and UI styling
- Sample school data (`data/`)

## Usage
- Open `index.html` in a browser
- Images/CSS/JS are referenced from the same directory and `assets`

## Structure
- `index.html` etc.: page templates
- `style.css`: global styles
- `script.js`: helper scripts for pages
- `assets/`: images and brand assets
- `data/`: sample data

## Status
- UI/flow mock (implementation TBD)
