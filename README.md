# CanadaClassi

CanadaClassi はカナダ向けのクラシファイド（掲示板）Web UI モックです。
静的HTML/CSS/JSで構成され、主要ページの画面設計・導線・UIコンポーネントを確認できます。

## 内容
- 一覧、詳細、投稿、プロフィール、問い合わせなどの主要画面
- ブランドアセットとUIスタイル
- 学校データのサンプル（`data/`）

## 使い方
- `index.html` をブラウザで開く
- 画像・CSS・JSは同階層/`assets` を参照

## 構成
- `index.html` など: 画面テンプレート
- `style.css`: 全体スタイル
- `script.js`: 画面用の補助スクリプト
- `assets/`: 画像・ブランド素材
- `data/`: サンプルデータ

## ステータス
- 画面/導線の検討用モック
